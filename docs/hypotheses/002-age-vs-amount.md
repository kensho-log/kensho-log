# 仮説 002: 開始年齢と拠出額の対決

## 問い

1960〜1985 年の 26 通りの開始年（条件A）/ 1980〜2005 年の 26 通りの開始年（条件B）において、以下2条件の 60歳時点の最終資産はどちらが多いか。

- **条件A**: 20歳開始、毎月 3万円積立、40年間、総投資額 1,440万円
- **条件B**: 40歳開始、毎月 10万円積立、20年間、総投資額 2,400万円

## 指標

**60歳時点の最終資産額のみ**（ドル建て評価 → 円建て併記）

## 検証条件

| 項目 | 値 |
|---|---|
| 対象指数 | S&P 500 Total Return Index |
| 条件A 期間 | 開始年 + 40年 |
| 条件B 期間 | 開始年 + 20年 |
| 開始年の範囲 | 1960〜1985（26パターン） |
| 積立額A | 月3万円（年36万円） |
| 積立額B | 月10万円（年120万円） |
| 総投資額A | 1,440万円 |
| 総投資額B | 2,400万円 |
| 税制 | 税引前（NISA 成長枠想定） |
| 為替 | 現時点の為替レートで円換算。時系列為替影響は別動画 |
| データソース | 一次: yfinance `^SP500TR` / クロスチェック: `^GSPC` + Stooq `^SPX` |

## データ範囲の限界

- 1960年以前のデータが必要なパターンは除外（`^SP500TR` は 1988 年以降のため、古い期間は `^GSPC` + 配当利回り別データで合成）
- 為替: 条件 A/B の開始時期が異なるため、単純ドル建て比較を主とし、円換算は補助として提示
- 税制: 日本の配当課税・売却益課税は考慮しない

## 期待される結果

仮説:

- **中央値で A > B**（複利の非線形性が総投資額差 1,000万円を逆転）
- ただし分布のばらつきが大きく、特定期間（例: 2000-2020開始）では B > A もあり得る
- A と B の勝率は 60〜75% 程度で A 勝利想定（実測で確定）

## 反常識要素

- 視聴者の直感: 「総投資額が多い方（B）が勝つ」
- 計算結果の予測: **総投資額が少ない 20歳月3万円の A が、多数の期間で勝つ**
- 「複利 > 投資額」の数量化

## 視聴者予測プロンプト

> 総投資額が 1,000万円多い条件B（40歳月10万）が、60歳時点で条件A（20歳月3万）に勝つと思いますか？勝つ場合、何%上回ると思いますか？

## 動的描画仕様

- **オドメーター**: 各時点の累積資産額を桁区切り回転表示（A/B 並走）
- **スモールマルチプル**: 26パターンの結果を 6x5 グリッドで同時表示（最終静止画）
- **X軸**: 年齢単位（20歳〜60歳 / 40歳〜60歳）で固定
- **Y軸**: 0 〜 想定最大値の 1.1倍で固定

## 採否基準判定

- 中央値差分 ≥ 10% → 長尺公開
- 中央値差分 5〜10% → Shorts
- 中央値差分 < 5% → リポジトリログのみ

## ペルソナ表記運用

引継ぎ文書「ペルソナ運用」で通常動画は1ペルソナ固定ルールだが、本仮説は **開始年齢と拠出額そのものが検証変数** のため、ペルソナ表記ではなく **「条件A / 条件B」** の比較として扱う。

タイトル・概要欄でも「ペルソナ」の語は使用せず、「条件」表記で統一。
