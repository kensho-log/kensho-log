# 仮説 003: ドルコスト積立の利益化までの最悪年数

## 問い

1990年1月〜2024年12月 の各月を開始月として、毎月 3万円の S&P500 積立を始めた場合、**保有資産が元本を上回る（= 利益が出る）状態になるまでの年数**の分布はどうなるか。特に、**「絶対に利益が出る」と言える最悪ケースは何年か**。

## 指標

**利益化（保有資産評価額 ≥ 累積投資額）までの保有年数** の分布

- 中央値 / 最大値（最悪ケース）/ 最小値
- 開始月ごとの横軸散布図

## 検証条件

| 項目 | 値 |
|---|---|
| 対象指数 | S&P 500 Total Return Index |
| 期間 | 1990-01 〜 2024-12（34年） |
| 開始月 | 1990-01 から 2024-12 までの全月（420パターン） |
| 評価終期 | 2024-12 または「利益化達成」時点の早い方 |
| 積立額 | 月3万円（ペルソナ B の設定を流用） |
| データソース | 一次: yfinance `^SP500TR` / クロスチェック: `^GSPC` + Stooq `^SPX` |
| 為替 | ドル建てで検証。円換算は参考として末尾追加 |
| 税制 | 税引前 |

## データ範囲の限界

- 2024年末に近い開始月（例: 2020年以降開始）は、評価期間が短いため「まだ利益化していない」可能性がある
- その場合、結果は「2024年末時点で利益化未達」としてラベル付けし、保有年数の最大値からは除外する（別カテゴリで集計）
- 1990年より前のデータは対象外

## 期待される結果

仮説:

- **中央値**: 1〜3年程度で利益化（インデックス投資の一般的な感覚に近い）
- **最悪値**: リーマンショック直前（2007年10月頃）開始のケースが最大保有年数（想定 6〜8年）
- **通説「20年保有で元本割れしない」との対比**: 本検証は「最悪でも何年で利益出るか」。20年保有の元本割れ確率ゼロ説は部分的に裏付けされる

## 反常識要素

- 視聴者の直感: 「ドルコスト平均法なら 10〜20年は覚悟」
- 計算結果の予測: **最悪ケースでも 10 年以内に利益化**
- 逆の可能性: リーマン級の暴落直前から始めた場合、想像より長くかかる可能性（視聴者の直感「10年で十分」への反証）

どちらに転んでも反常識要素が立ち上がる設計。

## 視聴者予測プロンプト

> 過去34年間の中で、毎月 3万円を S&P500 に積立し始めて、利益が出るまで最も長くかかったケースは何年だと思いますか？

## 動的描画仕様

- **段階プロット**: 横軸=開始月（1990-01 〜 2024-12）、縦軸=利益化までの年数。420点が時系列順にプロット
- **具体例アニメ**: 好期間・悪期間の2シナリオで累積資産推移を並走表示
- **最悪ケース強調**: 散布図で最大値の点を赤色ハイライト + 開始月をラベル表示
- **X軸固定**: 1990-01 〜 2024-12
- **Y軸固定**: 0 〜 想定最大値（15年程度）

## 採否基準判定

本仮説は「通説への部分的な反証 or 裏付け」のどちらに転んでも公開価値があるが、採否判定の基準は以下:

- 最悪値が視聴者予測の中央値（想定 10〜20年）から ±5年以上ズレ → 長尺公開
- ±2〜5年のズレ → Shorts
- ±2年以内（予測通り） → リポジトリログのみ

## ペルソナ

ペルソナ B（月3万円 / 30歳開始 / NISA つみたて枠）準拠。  
ただし本検証では「30歳」という年齢は本質変数ではなく、開始月そのものが変数。タイトル・概要欄では「月3万円の積立投資家」の表現で統一し、ペルソナ記号は使用しない。
