# 仮説 001: タイミング3者レース

## 問い

1990年から2024年までの34年間、以下3条件の投資家の最終資産は何%の差が出るか。

- **条件A**: 毎年、S&P500 が**最安値**を記録した月に 100万円を一括投資する（完璧タイミング投資家）
- **条件B**: 毎年、1月から12月まで均等に分割し、毎月 83,333円を投資する（毎月ドルコスト投資家）
- **条件C**: 毎年、S&P500 が**最高値**を記録した月に 100万円を一括投資する（最悪タイミング投資家）

## 指標

**最終資産額のみ**（2024年12月末時点の保有資産評価額、配当再投資込み）

CAGR・最大DD・Sharpe は扱わない（1動画1指標ルール）。

## 検証条件

| 項目 | 値 |
|---|---|
| 対象指数 | S&P 500 Total Return Index |
| 期間 | 1990-01 〜 2024-12（34年） |
| 年間投資額 | 100万円（全条件で同額） |
| 総投資額 | 3,400万円 |
| 通貨 | 円建て想定（為替影響は別途明記） |
| 税制 | 税引前（複数税制検証は別動画で扱う） |
| データソース | 一次: yfinance `^SP500TR` / クロスチェック: yfinance `^GSPC` + Stooq `^SPX` 価格成分 |
| 配当 | 再投資込み（`^SP500TR` が配当再投資を内包） |

## データ範囲の限界

- 為替: 円建て換算を行う場合、為替ヒストリカルデータの取得源を明記し、ドル建て結果と併記する
- 税制: 日本の配当課税（20.315%）・売却益課税は考慮しない
- 取引コスト: 信託報酬・スプレッドは考慮しない

## 期待される結果

仮説（事前の見立て）:

- **A > B > C** の順で最終資産が並ぶ
- **A と C の差**: +30〜50% 程度
- **A と B の差**: +5〜15% 程度
- **C でも元本 3,400万円は大幅プラス**（視聴者の直感は「最悪タイミングなら損する」）

## 反常識要素

- 視聴者の直感: 「最高値で買い続けたら、30年超でも損が残る」
- 計算結果の予測: 最悪タイミング投資家 C でも**元本対比で +○%（実測で確定）**の利益が出る
- 「タイミングを気にするより、継続すること」の定量化

## 視聴者予測プロンプト
